# Cursos Online - Landing Page

Landing page de una plataforma de cursos online, desarrollada íntegramente con **Bootstrap 5** como ejercicio práctico para consolidar conocimientos sobre el framework.

## Objetivo

Reforzar y afianzar conceptos clave de **Bootstrap 5**:
- Sistema de **grid** (`container`, `row`, `col-*`, `row-cols-*`)
- Componentes: **Navbar**, **Cards**, **Badges**, **Buttons**
- Clases de **utilidad** (tipoografía, espaciado, flexbox, display, border-radius)
- **Responsividad** con breakpoints (`col-md-*`, `d-md-*`, `gap-md-*`)
- **Personalización** de temas mediante variables CSS propias combinadas con clases de Bootstrap

## Secciones

- **Inicio** — Hero section con presentación llamativa y fondo personalizado.
- **Nosotros** — Información sobre la plataforma y skills destacados.
- **Cursos** — Catálogo de cursos populares con tarjetas (cards).

## Tecnologías

- HTML5
- CSS3 (con variables personalizadas)
- Bootstrap 5.3.8
- Bootstrap Icons
- Google Fonts (Elms Sans)

## Cómo usar

Abrí `index.html` en tu navegador o serví la carpeta con un servidor local:

```bash
npx live-server
```
